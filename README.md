# react-wrappy-text
Wrappy text component for React

## Demo

http://dkozar.github.io/react-wrappy-text/

## Installation

```bash
git clone https://github.com/dkozar/react-wrappy-text.git
cd react-wrappy-text
npm install
npm start
open http://localhost:3000
```
